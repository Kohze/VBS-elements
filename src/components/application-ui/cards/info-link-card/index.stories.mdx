# Info Link Card

Info link card is a card that displays basic information about a product or service with a link.

## Import

```js
import { InfoLinkCard } from '@/components/application-ui/cards'
```

## Default

To use the default info link card, simply import the component and pass in the required props `title`, `description`, `iconName` and `link`. You can customize the card with using props. See the [reference table](#reference) for a list of props.
