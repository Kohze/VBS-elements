# ContactCard

ContactCard is a card that displays user basic information.

## Import

```js
import { ContactCard } from '@/components/application-ui/cards'
```

## Default

To use the ContactCard, simply import the component and pass the user object as a prop. User object should have the following properties: `name`, `title`, `role`, `email`, `phone`, `imageSrc`.
