## Stacked List

Stacked list is a list of items that are stacked on top of each other. It is used to display a list of items that are related to each other.

## Import

```js
import { StackedList } from '@/components/application-ui/lists'
```

## Simple

Simple stacked list uses `InfoItem` as default item component. `items` prop is required. `items` prop is an array of objects with same keys with `InfoItem` props to create list items. Check `InfoItem` documentation for props information [here](/docs/application-ui-list-items-info-item--default-story)

## Link Item

Add `href` property inside `items` object to make the item clickable. `href` property is a string that will be used as `href` attribute of `a` tag.

Also with `iconName` prop, you can add an icon to the item. `iconName` prop is a string that will be used as `iconName` prop of `Icon` component. Check `Icon` documentation for props information [here](/docs/application-ui-elements-icon--sample)
