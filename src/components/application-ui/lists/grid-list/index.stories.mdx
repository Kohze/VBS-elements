# Grid List

This is not a component, this is just show how to use the Grid component with the various components.

## Contact Cards Grid List

Usage Grid with ContactCard component. Check the ContactCard component for more details about the props [here](/docs/application-ui-cards-contact-card--default-story).

### Horizontal

<Preview>
  <div className="p-4 bg-slate-50">
    <Grid columnsLg={3} columnsSm={2}>
      {contactUsers.map((user) => (
        <ContactCard key={user.id} user={user} avatarSize="md" />
      ))}
    </Grid>
  </div>
</Preview>

### Vertical

<Preview>
  <div className="p-4 bg-slate-50">

  </div>
</Preview>

## Info Link Cards Grid List With Shared Borders

Usage Grid with InfoLinkCard component. Check the InfoLinkCard component for more details about the props [here](/docs/application-ui-cards-info-link-card--default-story).

For merging css classes, we use the twMerge function from the tailwind-merge package. To use it;

```js
import { twMerge } from '@tailwindcssinjs/tailwind-merge'`
```

<Preview>
  <div className="py-4 px-12 bg-slate-200">
    <Grid columnsSm={2} gap="px">
      {infoLinkItems.map((item, actionIdx) => (
        <InfoLinkCard
          key={item.id}
          {...item}
          iconSize="md"
          classNames={{
            main: twMerge(
              'rounded-none border-none hover:border-none',
              actionIdx === 0
                ? 'rounded-tl-lg rounded-tr-lg sm:rounded-tr-none'
                : '',
              actionIdx === 1 ? 'sm:rounded-tr-lg' : '',
              actionIdx === infoLinkItems.length - 2 ? 'sm:rounded-bl-lg' : '',
              actionIdx === infoLinkItems.length - 1
                ? 'rounded-bl-lg rounded-br-lg sm:rounded-bl-none'
                : '',
            ),
          }}
        />
      ))}
    </Grid>
  </div>
</Preview>
