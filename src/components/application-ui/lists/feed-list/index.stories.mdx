# Feed List

Feed list is a list of items that are displayed in a vertical list. It is used to display a list of feeds.

## Import

```js
import FeedList from '@/components/application-ui/lists/feed-list'
```

## Default

Feed list takes `timeline` as a prop. `timeline` is an array of objects that contains the following properties: `text`, `href`, `date`, `iconName`, `iconBg`, `iconColor`. Check [Heroicons](https://heroicons.com/) for the list of icon names. `iconColor` takes a hex color code. `iconBg` takes a tailwindcss color class.
