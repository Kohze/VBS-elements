# Tab

Tabs are used to organize content into different sections. They are useful when you have a lot of content and need to organize it into different sections.

## Default

<Canvas>
  <Story name="Default">{Default.bind({})}</Story>
</Canvas>

## With Icon

Tabs can also have icons. `iconName` is the name of the icon. `iconPosition` is the position of the icon. It can be `left` or `right`.

Check the icon names in the [Heroicons](https://heroicons.com/) library.

<Canvas>
  <Story name="With Icon">{WithIcon.bind({})}</Story>
</Canvas>

## With Pills

Tabs can also be styled as pills. `variant` can be `default` or `pills`.

<Canvas>
  <Story name="With Pills">{WithPills.bind({})}</Story>
</Canvas>

## Custom Tab Item Styles

You can also customize the styles of the tab items. `tabItemClassName` is the class name of the tab item. `currentClassName` is the class name of the current tab item.

<Canvas>
  <Story name="Custom Tab Item Styles">{CustomTabItemStyles.bind({})}</Story>
</Canvas>

## Full Width

Tabs can also be full width. Simply pass `fullWidth` prop.

<Canvas>
  <Story name="Full Width">{FullWidth.bind({})}</Story>
</Canvas>

## Bar Kind

Tabs can also be styled as a bar. To do this, pass `kind="bar"`.

  <Story name="Bar Kind">{BarKind.bind({})}</Story>
</Canvas>

## With Tab Count

Tabs can also have a count. `count` is the number of the count.
