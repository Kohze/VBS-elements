# Tab

Tabs are used to organize content into different sections. They are useful when you have a lot of content and need to organize it into different sections.

## With Icon

Tabs can also have icons. The icon can be placed on the left or right side of the tab name. Check the icon names in the [Heroicons](https://heroicons.com/) library.
