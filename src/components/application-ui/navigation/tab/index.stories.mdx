# Tab

Tabs are used to organize content into different sections. They are useful when you have a lot of content and need to organize it into different sections.

## Import

```js
import { Tab } from '@/components/application-ui/navigation'
```

## With Icon

Tabs can also have icons. `iconName` is the name of the icon. `iconPosition` is the position of the icon. It can be `left` or `right`.

Check the icon names in the [Heroicons](https://heroicons.com/) library.

## Custom Tab Item Styles

You can also customize the styles of the tab items. Use the `classNames` prop object to pass the class names. `classNames.tabItem` is the class name of the tab item. `classNames.current` is the class name of the current tab item.

<Canvas>
  <Story
    name="Custom Tab Item Styles"
    args={{

          href: '#',
          current: false,

        },
        { name: 'Team Members', href: '#', current: true, iconName: 'users' },
        { name: 'Billing', href: '#', current: false, iconName: 'credit-card' },
      ],
      variant: 'pills',
      classNames: {
        tabItem: 'bg-white text-gray-900 rounded-full border border-gray-200',
        current: 'bg-indigo-600 text-white rounded-full',
      },
    }}
    parameters={{
      docs: {
        source: {
          type: 'code',
          code: `<Tab
  tabs={[
    { name: 'My Account', href: '#', current: false, iconName: 'user' },
    {
      name: 'Company',

      current: false,
      iconName: 'computer-desktop',
