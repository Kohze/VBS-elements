# Breadcrumb

Breadcrumb is a navigation component that shows the current pageâ€™s location within a navigational hierarchy.

Breadcrumbs needs to have a list of `pages` to display. Each page is an object with the following properties: `name`, `href` and `current`. The `current` property is used to indicate the current page.

## Import

```js
import { Breadcrumb } from '@/components/application-ui/navigation'
```
