# Info Item

Info Item is a component that displays information in a list.

## Simple

There are props to display the title and other values like avatar, description. Title can be a name or a label. To list sub information, use the `metaList` prop. The `metaList` prop is an array of objects with `name` and `iconName` properties. You can list single or multiple items in the `metaList` prop.

<Canvas>
  <Story
    name="Default"
    args={{
      title: 'John Taylor',
      metaList: [
        {
          name: 'john@example.com',
          iconName: 'envelope',
        },
      ],
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## With Avatar

Displaying an avatar is optional. You can use the `withAvatar` prop to display an avatar. If you dont give `imageSrc`, it will display placeholder image. Check Avatar component for more details [here](/docs/application-ui-elements-avatar--default-story).

There is also `avatarPosition` prop to display the avatar on the left, right, top or bottom of the title. The default value is `left`. You can use `avatarSize` prop to change the size of the avatar. The default value is `lg`.

<Canvas>
  <Story
    name="With Avatar"
    args={{
      title: 'Calvin Hawkins',
      withAvatar: true,
      metaList: [

          name: 'john@example.com',
          iconName: 'envelope',
        },
      ],
    }}
  >
    {(args) => (
      <>
        <VBSInfoItem {...args} />

          {...args}

          imageSrc="https://images.unsplash.com/photo-1500648767791-00dcc994a43e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=facearea&facepad=2&w=256&h=256&q=80"
        />
      </>
    )}
  </Story>
</Canvas>

## With Custom Icon Color and Position

You can use the `iconColor` and `iconPosition` props to customize the icon color and position.

<Canvas>
  <Story
    name="With Custom Icon Color and Position"
    args={{
      title: 'Applied on Jan 1, 2020',
      metaList: [
        {
          name: 'Completed',
          iconName: 'check-circle',
        },
      ],
      iconColor: '#00B87C',
      iconPosition: 'right',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## With Actions

You can use the `actions` prop to display actions. The `actions` prop is an array of objects with `VBSButton` props. Check VBSButton component for more details [here](/docs/application-ui-elements-button--default-story).

<Canvas>
  <Story
    name="With Actions"
    args={{
      title: 'Calvin Hawkins',
      withAvatar: true,
      metaList: [
        {
          name: 'calvin.hawkins@example.com',
          iconName: 'envelope',
        },
      ],
      actions: [
        {
          iconName: 'pencil',
          text: 'Edit',
          onClick: () => alert('Edit'),
          size: 'xs',
          variant: 'outline',
        },
        {
          iconName: 'trash',
          text: 'Delete',
          onClick: () => alert('Delete'),
          size: 'xs',
          variant: 'outline',
        },
        {
          iconName: 'eye',
          text: 'Link',
          href: 'https://www.google.com',
          size: 'xs',
          variant: 'outline',
        },
      ],
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
