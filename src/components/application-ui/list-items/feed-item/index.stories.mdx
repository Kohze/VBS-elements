# Feed Item

Feed Item is a component that displays a feed item inside a feed list.

## Import

```js
import { FeedItem } from '@/components/application-ui/list-items'
```

## Comment Variant

To use the comment variant, use the `variant` prop and set it to `comment`.

<Canvas>
  <Story
    name="Comment Variant"
    args={{
      variant: 'comment',
      person: { name: 'Eduardo Benz', href: '#' },
      imageSrc:
        'https://images.unsplash.com/photo-1520785643438-5bf77931f493?ixlib=rb-=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=facearea&facepad=8&w=256&h=256&q=80',
      comment:
        'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Tincidunt nunc ipsum tempor purus vitae id. Morbi in vestibulum nec varius. Et diam cursus quis sed purus nam. ',
      commentIconName: 'chat-alt-2',
      time: '6d ago',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Assignment Variant

To use the assignment variant, use the `variant` prop and set it to `assignment` and `assigned` prop with the assigned person.

<Canvas>
  <Story
    name="Assignment Variant"
    args={{
      variant: 'assignment',
      person: { name: 'Hilary Mahy', href: '#' },
      assigned: { name: 'Kristin Watson', href: '#' },
      time: '2d ago',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Tags Variant

To use the tags variant, use the `variant` prop and set it to `tags` and `tags` prop with the tags array including the name, href and color.

<Canvas>
  <Story
    name="Tags Variant"
    args={{
      variant: 'tags',
      person: { name: 'Hilary Mahy', href: '#' },
      tags: [
        { text: 'Bug', href: '#', color: '#f43f5e' },
        { text: 'Accessibility', href: '#', color: '#4f46e5' },
      ],
      time: '6h ago',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
