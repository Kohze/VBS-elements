# Grid

The Grid component is a grid system that allows you to easily create a grid layout.

For interact with the component, you can use the controls and see the results with viewport addon. You can see the viewport addon in the panel. Indicated in screenshot below.

<p>
  <img src={viewportAddon} alt="Viewport addon" />
</p>

## Usage

You can use cards, divs, or any other element as `children` of the Grid component. There are props `columns`, `columnsSm`, `columnsMd`, `columnsLg` and `columnsXl` to change the number of columns for each breakpoint. There are also some ready gap sizes to use with the `gap` prop. Check reference for more details.

For more customization, you can use the tailwind classes to change the grid layout. Check the [Tailwind CSS Grid](https://tailwindcss.com/docs/grid-template-columns) for more details.

## Default

Default grid is 1 column for all breakpoints.
