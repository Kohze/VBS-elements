# List Container

The list container is a container that can be used to display a list of items.

For interact with the component, you can use the controls and see the results with viewport addon. You can see the viewport addon in the panel. Indicated in screenshot below.

<p>
  <img src={viewportAddon} alt="Viewport addon" />
</p>

## Import

```js
import { ListContainer } from '@/components/application-ui/layout'
```

## Variants

List Container has 7 variants: `simple`, `card-with-divider`, `card-with-divider-full`, `separated-cards`, `separated-cards-full`, `flat-with-divider` and `simple-with-divider-full`
