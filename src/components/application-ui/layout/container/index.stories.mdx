# Container

The container component is a wrapper for the content of the page. It is used to center the content and to set the maximum width of the content.

Default container is full width with padding.

For interact with the component, you can use the controls and see the results with viewport addon. You can see the viewport addon in the panel. Indicated in screenshot below.

<p>
  <img src={viewportAddon} alt="Viewport addon" />
</p>


## Narrow Padded

Narrow constrained with padded content.

There are few narrow variants of the container component. For changing the width of the container, you can use the `narrowSize` prop. Default value is `3xl`. You can use `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `7xl`, `full`.
