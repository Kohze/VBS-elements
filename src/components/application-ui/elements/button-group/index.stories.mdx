## Button Group

Button groups are used to group buttons together.

## Import

```js
import { ButtonGroup } from '@/components/application-ui/elements'
```

## Basic

Basic group of buttons. The `elements` prop is an array of objects with the following properties: `text` and `iconName` (optional). Almost every property of the `Button` component can be used in the `ButtonGroup` component. Check the reference below.
