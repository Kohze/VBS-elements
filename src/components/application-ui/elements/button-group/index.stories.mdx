## Button Group

Button groups are used to group buttons together.

## Import

```js
import VBSButtonGroup from '@/components/application-ui/elements/button-group'
```

## Basic

Basic group of buttons. The `groupElements` prop is an array of objects with the following properties: `text` and `iconName` (optional). Almost every property of the `VBSButton` component can be used in the `VBSButtonGroup` array. Check the reference below.
