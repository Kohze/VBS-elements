# Avatar

Avatar is a component that displays a user's profile picture.

## Import

```js
import VBSAvatar from '@/components/application-ui/elements/avatar'
```

## Usage

Avatar component takes a `imageSrc` prop which is the url of the image to be displayed.

## Sizes

Avatar component takes a `size` prop which is the size of the avatar. The size prop can be one of the following: `xs` `sm`, `md`, `lg`, `xl`. If no size is provided, the default size is `md`.

<Canvas>
  <Story name="Sizes">{Sizes.bind({})}</Story>
</Canvas>

## Kinds

Avatar component takes a `kind` prop which is the shape of the avatar. The kind prop can be one of the following: `circular`, `rounded`, `square`. If no variant is provided, the default kind is `circular`.

<Canvas>
  <Story name="Kinds">{Kinds.bind({})}</Story>
</Canvas>

## Variants

There are 3 variants of avatar. If user provides an image, the avatar will be user image. If user doesnt provide an image, the avatar will be the first letter of the name. If both image and name are not provided, the avatar will be a placeholder avatar.

<Canvas>
  <Story name="Variants">{Variants.bind({})}</Story>
</Canvas>

## With Notification

Avatar component takes a `withNotification` prop which is a boolean value. If true, a notification dot will be displayed on the avatar. The `notificationPosition` prop can be one of the following: `topLeft`, `topRight`, `bottomLeft`, `bottomRight`. If no notificationPosition is provided, the default notificationPosition is `topRight`. There is also a `notificationColor` prop which is the color of the notification dot.

</Canvas>
