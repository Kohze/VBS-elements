# Button

Buttons allow users to take actions, and make choices with a single tap.

## Import

```js
import Icon from '@/components/application-ui/elements/icon'
```

## Usage

- A button should contain at least one text, icon or both.
- Button texts are limited to one line.
- Buttons can be used to trigger an action, or to navigate to a new page with href attribute.
- For button colors, `backgroundColor` and `color` props or tailwind classes can be used.
- For button sizes, `size` prop or tailwind classes can be used.
- All button styles can be customized with tailwind classes.
