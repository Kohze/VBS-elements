# Avatar Stack

The Avatar Stack component is used to display a list of avatars in a stack.

## Import

```js
import VBSAvatarStack from '@/components/application-ui/elements/avatar-stack'
```

## Usage

To use the Avatar Stack component, simply import the component and pass array of items to the `items` prop with `name` and `imageSrc` properties.
