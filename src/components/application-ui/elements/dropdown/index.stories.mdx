# Dropdown

Dropdowns are used to display a list of options that can be selected by the user.

## Default

Dropdowns need to have `elements` props containing an array of objects with `href`, `text` and `iconName` (optional) properties.

<Canvas>
  <Story
    name="Default"
    args={{
      elements: [{ text: 'Link 1' }, { text: 'Link 2' }, { text: 'Link 3' }],
      buttonText: 'Dropdown',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## With Icon

Dropdowns can also contain icons. The icon can be placed on the left or right side of the text with the `listIconPosition` prop. Use the `iconName` property to specify the icon. Icons names can be found in the [Heroicons](https://heroicons.com/) library.

<Canvas>
  <Story
    name="With Icon"
    args={{
      elements: [
        { text: 'Link 1', iconName: 'academic-cap', href: '#' },
        { text: 'Link 2', iconName: 'adjustments-vertical', href: '#' },
        { text: 'Link 3', iconName: 'arrow-up-tray', href: '#' },
      ],
      buttonText: 'Dropdown',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Grouped Elements

Dropdowns can also contain grouped elements. The `groupElements` prop is used to group elements together. The `groupElements` prop is an array including the number of arrays of objects that will be grouped together. Each object must have a `text` and `iconName` (optional) property.

<Canvas>
  <Story

## Minimal

Minimal dropdowns are have a transparent background and a borderless button. Button icon can change with the `buttonIconName` prop, check the other props in the Reference section.

Dropdowns can have different ready-to-use variants. The `variant` prop can be used to change the dropdown variant. Check the other props in the Reference section.
