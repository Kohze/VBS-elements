# Badge

Badges are used to highlight important information.

## Import

```js
import { Badge } from '@/components/application-ui/elements'
```

## Default

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>
</Canvas>

## Variants

Variants are used to highlight the importance of the badge. The default variant is `primary`. The other variants are `secondary`, `success`, `danger`, `warning`, `info`, `outline`, `light`, and `dark`.

<Canvas>
  <Story name="Variants">{Variants.bind({})}</Story>
</Canvas>

## Kinds

Kinds are used to change the shape of the badge. The default kind is `rounded`. The other kinds are `square` and `circular`.

<Canvas>
  <Story name="Kinds">{Kinds.bind({})}</Story>
</Canvas>

## Sizes

Sizes are used to change the size of the badge. The default size is `normal`. The other size is `small`.

<Canvas>
  <Story name="Sizes">{Sizes.bind({})}</Story>
</Canvas>

## With Notification

Badges can have a notification dot with `notification` prop.

  <Story name="With Notification">{WithNotification.bind({})}</Story>
</Canvas>
