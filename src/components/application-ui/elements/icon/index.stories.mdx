# Icon

Icon component for displaying icons. The icon component is a wrapper for the [Hero Icons](https://heroicons.com/) library.

## Import

```js
import { Icon } from '@/components/application-ui/elements'
```

## Usage

To use the icon component, simply pass the `iconName` as a prop. The `iconName` prop is the name of the icon you want to display. The icon name is the same as the icon name in the [Hero Icons](https://heroicons.com/) library.

The icon component supports outline icons. To use an outline icon, pass the `iconType` prop with the value `outline`.

## Sizes

The icon component supports different sizes. To use a different size, pass the `size` prop with the value `xs`, `sm`, `md`, `lg`, or `xl`.
