# Tag

Tags are used to label or categorize content. They are used to indicate status, priority, or type.

## Import

```js
import Tag from '@/components/application-ui/elements/tag'
```

## Default

Default tag icon is a circle.

<Canvas>
  <Story name="Default">
    <Tag text="Default" />
  </Story>
</Canvas>

## With Icon

If you want to add an icon, you can use the `iconName` prop.

<Canvas>
  <Story name="With Icon">
    <Tag text="With Icon" iconName="check" />
  </Story>
</Canvas>

## Variants

Variants are used to highlight the importance of the tag. The default variant is `outline`. The other variants are `secondary`, `success`, `danger`, `warning`, `info`, `outline`, `light`, and `dark`.

## Sizes

There are 3 sizes available: `sm`, `md`, and `lg`. The default size is `md`.

## Kinds

There are 3 kinds available: `square`, `circular`, and `rounded`. The default kind is `circular`.
