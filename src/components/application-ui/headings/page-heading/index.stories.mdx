# Page Heading

The page heading is used to display the title of a page.

Some parts of the page heading are optional, such as the action buttons. To add some part of as node, you can use the `render props` pattern.

## Import

```js
import VBSPageHeading from '@/components/application-ui/headings/page-heading'
```
