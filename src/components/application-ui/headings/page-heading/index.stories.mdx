# Page Heading

The page heading is used to display the title of a page.

Some parts of the page heading are optional, such as the action buttons. To add some part of as node, you can use the `render props` pattern.

## Import

```js
import VBSPageHeading from '@/components/application-ui/headings/page-heading'
```

## With Avatar

Heading with avatar need `user` prop. The `user` prop is an object with `name`, `role`, `imageSrc` and `link` properties. The `avatarSize` prop is used to set the size of the avatar, `description` prop is used to set the description of the user. There are also `avatarKind` and `avatarVariant` props to set the kind and variant of the avatar.

<Canvas>
  <Story name="With Avatar">{WithAvatar.bind({})}</Story>
</Canvas>

## Card View

Heading with card view need `user` and `cardList` props. The `user` prop is an object with `name`, `role`, `imageSrc` and `link` properties. The `cardList` prop is an array of object with `label` and `value` properties.

<Canvas>
  <Story name="Card View">{CardView.bind({})}</Story>
